Создание многомерного контейнера - сетки ( display: grid;)

  1. Сбрасываем базовые настройки в css.
    * {
    margin: 0;
    padding: 0;
    }
  2. Создаём общий контейнер- который будет в себе хранить и размещать все элементы на веб странице. Для этого необходим тег div и класс ему :
    div class="conteiner"
  3. В главном контейнере разметим - блок с картой товара :
    div class="cart"
  4. Разместим внутри изображения в теге div.cart -> img src="" alt=""- Картинка будет очень большого размера . Временнно сделаем ей ширину 100px- чтоб стало более наглядно как всё это будет работать

    Ранее рассматривали контейнер flex ( ОДНОМЕРНЫЙ )- с одномерным распределением содержимого в горизонтальной или вертикальной плоскости.

  5. Обращаемся к главному контейнеру и ставим
    .conteiner { display : grid;}

    display: grid ;-(МНОГОМЕРНЫЙ) -Позволяет расположить элементы при помощи сетки. Сразу можно опреировать и строками и столбцами. Можем расположить как хочется элементы при помощи сетки

Оформление сетки (столбцы и ряды) grid-template-columns/ grid-template-rows

  1. Указываем сколько строк и колонок необходимо на нашей странице. Сделаем 3 колонки , а строк будет столько, сколько будет карточек товара.
    Для этого увеличим колличество карт к прримеру 10 шт.
    div class="cart">
    img src="static/images/лого.PNG" alt="Картинка логотипа"
  2. grid-template-columns: 1fr 1fr 1fr;
    Запишим спец свойства , которые управляют какая будет сетка
    grid-template-columns: укажем сколько будет столбцов (3) ;

    Если хотим 3 столбца то надо разделить сколько долей каждый будет занимать. Ширину экран поделить на 3 равные части (Можно 1 узкую и 2 широких)

    Чтоб написать 3 одинаковых доли- надо написать три значения этих долей.
    Доли обозначаются при помощи букв fr

  3. Если надо вручную задать колличество строк , которое долно быть пишим grid-template-rows: 500px 500px 500px (имеется ввиду не ширина а ВЫСОТА)
    Если записать 1-й цифрой приенится только к первой строке
    Если записать grid-template-rows: 500px 500px 500px;- применится ко всем строкам
    Есть специальная функция , которая позволяет продублровать однотипные действия repeat (значение- сколько строк (3), какая высота (500px))
    grid-template-rows: repeat(4, 500px);
    Чтоб стало красивее меняем ширину картинки на 300px.
    img { width: 300px; }
    Картинка img находится в контейнере div(class=conteiner) с запрогранимиранной шириной , значит мы можем записать ширину не в px, а как 100% от родительского контейнера.
    В данном случае с шириной так работаем (в %) - как с величиной измерения
    img { width: 100%; }
    Когда уыеличиваем манитор или уменьшаем - картинки не переноситятся, они уменьшаются вместе с экраном - пропорционально
  4. Начинаем исправлять : там где делаем строки убираем размерность и говорим повторить 4 строки и тогда в высоту строка подстроится под содержимое контейнера grid-template-rows: repeat(x) => Картинка получится
    grid-template-rows: repeat(4);
  5. Делаем зазоры.
